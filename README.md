
# Pixel Colony God

A pixel sandbox simulation where you can draw and interact with elements like Sand, Water, Fire, Lava, and more.

[![Built With pollinations.ai](https://img.shields.io/badge/Built%20With-pollinations.ai-blueviolet)](https://pollinations.ai)

## Features

- Draw particles: Sand, Water, Fire, Ice, Steam, Lava, Stone, Wood, Plant, Acid
- Adjustable gravity, temperature, brush size/mode
- **AI Image Generator** powered by [pollinations.ai](https://pollinations.ai) (model: `flux`)

## Getting Started

Install dependencies:
```
npm install
```

Run the app:
```
npm run dev
```

## AI Images

This app uses the [pollinations.ai](https://pollinations.ai) API to generate images directly in the browser — no API key required.
Image model: **flux**
